## 스프링 부트를 이용한 영화 리뷰 사이트 만들기
#### 책: '코드로 배우는 스프링 브트 웹 프로젝트' Part 4부분을 이용하였습니다.
#### 사용 기술: 스프링 부트, MariaDB, Spring Data JPA , Javascript, JQuery
#### 저번 Board와 가장 큰  차이점은 이번 영화 리뷰 사이트에서는 사진을 등록할 수 있는 점입니다.



#####영화 게시물을 올릴 떄 사진을 첨부할 수 있습니다. (사진 첨부 기술은 Jquery의 Ajax을 이용하였습니다.)
